Typezy Quickstart

Essential patterns for runtime type checks and assertions.

Installation

Typezy is available on npm and works beautifully with TypeScript out of the box.

# Using npm
npm install typezy

# Using pnpm
pnpm add typezy

Pattern 1: Type Guards

Use is* functions when you want to conditionally handle different types or narrow the type for TypeScript without throwing an error.

import { isString, isNumber, isEmail } from 'typezy';

function handleInput(value: unknown) {
  if (isEmail(value)) {
    // TypeScript correctly narrows `value` to string
    sendWelcomeEmail(value);
  } else if (isNumber(value)) {
    // `value` is a number
    console.log(value * 10);
  } else {
    // Unknown type
    console.error("Invalid input provided.");
  }
}

Pattern 2: Type Assertions

Use assert* functions when your application requires the type to be correct to proceed safely, or you want to fail fast (e.g., API payloads, Configuration files).

import { assertPlainObject, assertString, assertNonEmptyArray } from 'typezy';

function processConfig(config: unknown) {
  // 1. Must be an object
  assertPlainObject(config, "Config must be an object");
  
  // 2. Must have a valid string API key
  assertString(config.apiKey, "Missing API Key string");
  
  // 3. Must have endpoints defined
  assertNonEmptyArray(config.endpoints);

  // Now safely proceed with the strongly-typed config
  initializeClient(config.apiKey, config.endpoints);
}

Pattern 3: Form Validation

When you need to collect errors for a UI instead of throwing an exception, use the validation modules.

import { createValidator, validateRequired, validateEmail, validateSchema } from 'typezy/validate';

// 1. Compose multiple validators into one
const validateEmailField = createValidator(
  validateRequired,
  validateEmail
);

// 2. Validate an entire object (Schema)
const schema = {
  email: [validateRequired, validateEmail],
  name: validateRequired
};

const result = validateSchema(formData, schema);
if (!result.valid) {
  setErrors(result.errors); // Map of field errors
}

Pattern 4: Frontend Optimization

Typezy includes essential tools for optimizing UI performance and handling async operations.

import { debounce, throttle, deepClone, sleep } from 'typezy';

// Debounce search calls
const onSearch = debounce((q) => fetchResults(q), 300);

// Safe deep clone for state updates
const newState = deepClone(oldState);

// Async pause
await sleep(500);
