Typezy — Quickstart Guide

Small, explicit helpers for runtime type checks and assertions.

Install

npm install --save typezy
# or
pnpm add typezy
# or
yarn add typezy

Basic usage

Import only what you need — the package is ESM-friendly and tree-shakable.

import { isEmail, assertNumber } from 'typezy'

if (isEmail('you@example.com')) {
  // valid
}

assertNumber(42) // throws on invalid

Common functions

  • Type checks: isString, isNumber, isArray
  • Validators: isEmail, isUUID, isIPv4
  • Assertions: assertString, assertNonEmptyArray

Example — Form validation

import { assertEmail, assertNonEmptyString } from 'typezy'

function submit(form) {
  assertEmail(form.email)
  assertNonEmptyString(form.name)
  // safe to proceed
}
