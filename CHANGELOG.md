# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-05

### Added
- Initial release
- **70+ type check functions** (`isString`, `isNumber`, `isEmail`, `isURL`, etc.)
- **13 assertion functions** (`assertString`, `assertNumber`, etc.)
- **String utilities** (`capitalize`, `toCamelCase`, `toKebabCase`, etc.)
- **Comparison functions** (`isEqual`, `isArrayEqual`, `hasChanged`, etc.)
- **Environment detection** (`isBrowser`, `isNode`, `getBrowserDetails`)
- **Form validation** (`isPhoneNumber`, `isCreditCard`, `isStrongPassword`)
- **File checks** (`isImageFile`, `isDocumentFile`, `isMimeType`)
- Full TypeScript support with type guards
- Zero runtime dependencies
- ESM and CommonJS builds
- 261 tests
